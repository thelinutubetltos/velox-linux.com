Why Velox

Everything you need, nothing you don't

Built on Arch with a focused set of sensible defaults.

⚡

Arch-based Rolling Release

Always up-to-date software from the Arch Linux repositories. Install once, update forever — no version upgrades needed.

🖥

Three Desktop Choices

Pick KDE Plasma, Cinnamon, or XFCE at install time. Each is fully themed and tuned for the Velox experience.

🌊

Wayland-first

All three desktops run on Wayland by default, giving you smooth rendering, HDR-ready pipelines, and better GPU integration.

🎨

Cohesive Velox Theme

A unified dark-green aesthetic — Fluent-Velox-Dark GTK theme, Newaita-Velox icon pack, and matching wallpapers across every desktop.

📦

Calamares Installer

Graphical partitioner with Btrfs subvolume support, timezone/locale setup, and per-desktop package selection built in.

🔐

Minimal Bloat

Only the tools you actually use. No bundled office suites, no trialware — just a clean base with Firefox and the essentials.

Desktops

Your environment, your way

Three fully-featured desktops, each tuned with Velox styling from the first boot.

💎

KDE Plasma

Wayland · Plasma 6

The flagship Velox desktop. Powerful, customizable, and beautiful — KDE Plasma 6 on Wayland with ksmoothdock and a hand-tuned panel layout.

  • Plasma 6 on Wayland
  • ksmoothdock launcher
  • Papirus-Green icon theme
  • Panel Colorizer integration
🌿

Cinnamon

Wayland · Cinnamon 6

The familiar, traditional layout on a modern Wayland stack. Nemo file manager, green-accented menus, and Fluent-Velox-Dark GTK theme throughout.

  • Cinnamon 6 on Wayland
  • Fluent-Velox-Dark GTK theme
  • Newaita-Velox icons
  • Green hover accents in Nemo
🪟

XFCE

Wayland · labwc compositor

Lightweight and fast. XFCE panels replaced with Waybar on top and Crystal Dock on the bottom, all styled with Velox dark-green colors.

  • labwc Wayland compositor
  • Waybar top panel
  • Crystal Dock bottom bar
  • Low memory footprint
